# Investment Intelligence Engine

محرك التحليل الاستثماري الذكي - نظام متكامل لتقييم الفرص الاستثمارية في السوق السعودي

## 🎯 المميزات الرئيسية

### 1. منهجية التحليل الخماسي المرجحة
- **التحليل الاقتصادي** (20%): حالة السوق، دورة القطاع، العرض والطلب
- **التحليل المالي** (30%): ROI، IRR، فترة الاسترداد، التدفقات النقدية
- **التحليل المقارن** (20%): أفضل 3 بدائل في السوق
- **التحليل القانوني** (15%): المخاطر التنظيمية، المستندات
- **تحليل الإنتاجية** (15%): الملاءمة، سهولة التسييل

### 2. نظام التعلم الآلي
- يحلل نتائج التحليلات السابقة
- يُحسّن أوزان المحركات تلقائياً
- يتكيف مع كل نوع أصل وهدف استثماري

### 3. بيانات السوق الحية
- أسعار الذهب اللحظية
- بيانات أسهم تداول
- مؤشر العقار للمدن الرئيسية
- مقارنة السعر المدخل بالسوق الفعلي

### 4. نظام التنبيهات الذكي
- تنبيه عند ارتفاع/انخفاض السعر
- تنبيه عند نسبة تغير محددة
- إشعارات فورية للمستخدم

## 🚀 التقنيات المستخدمة

- **Frontend**: React + TypeScript + Tailwind CSS + shadcn/ui
- **Backend**: Node.js + tRPC + Drizzle ORM
- **AI**: OpenAI GPT-4 للتحليل الاستثماري
- **Database**: MySQL
- **Auth**: Manus Auth

```

│   │   └── components/
│   └── index.html
├── server/
│   ├── routers.ts          # tRPC routers
│   ├── analysisEngine.ts   # محرك التحليل الخماسي
│   ├── learningEngine.ts   # نظام التعلم الآلي
```

## 🛠️ التشغيل محلياً

```bash
# تثبيت الاعتماديات

# تشغيل قاعدة البيانات
npm run db:push

# تشغيل التطبيق
npm run dev
```
